This rule detects insufficient javadoc (empty or containing only the @author tag)

Noncompliant Code Example

/**
 *
 * @author lazyman
 **/
class ClassWithInsufficientJavadoc {

    int aField;

}

Compliant Solution

/**
 * This class is responsible for something very important.
 *
 * @author braveman
 **/
class ClassWithJavadoc {

    int aField;

}
